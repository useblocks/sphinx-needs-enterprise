License
=======
``Sphinx-Needs Enterprise`` is released under the ``BSL 1.1``, the Business Source License.
Which mostly means the usage is free for private projects, but commercial projects needs to purchase a license.

It is also a ``Copyleft`` license, so that extensions using ``Sphinx-Needs-Extensions`` most be published under the same
BSL 1.1 and a commercial usage of the newly created extension os only possible with a purchased license.
